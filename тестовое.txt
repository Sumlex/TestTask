1. Написать программу, которая подсчитывает количество повторений каждого числа, встреченного в заданной строке.

Пример 1.

input: t5qr33d56q3
output:
5: 2
3: 3
6: 1


Пример 2.

input: ;f2*14'2444
output:
2: 2
1: 1
4: 4

2. Написать класс Cycler, принимающий список объектов при создании. 
У него должна быть функция, которая возвращает следующий объект из этого списка по порядку. 
Если мы достигли последнего элемента и ещё раз вызвали эту функцию, то должны вернуться в начало 
и снова пойти по кругу.

Пример использования псевдокодом.

list = { 1, 2, 3 }
cycler = Cycler(list)

do loop count = 8
    val = cycler.GetNext()
    print(val)

output: 1, 2, 3, 1, 2, 3, 1, 2
